Major API Endpoints

Accounts

/v2/accounts - Create, read, update, and delete accounts

Users

/v2/users - Manage end users and account users

Devices

/v2/devices - VoIP device configuration and management

Phone Numbers

/v2/phonenumbers - Search, order, port, and assign numbers

Call Routing

/v2/callforwarding - Configure forwarding and hunt groups

Voicemail

/v2/voicemail - Voicemail settings and greetings

Calling Plans

/v2/callingplans - Manage calling plan assignments

E911

/v2/e911 - Emergency address validation and registration

CDR

/v2/cdr - Call detail records and usage reports

Media

/v2/media - Upload audio files and hold music

IVR

/v2/ivr - Interactive voice response menus

Bulk Operations

/v2/bulk - Batch create/update operations
